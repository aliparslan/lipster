Do not add parallax effects, Ken Burns animation, or CoreMotion-based motion to the app.

**Why:** User explicitly said it makes them nauseous. The original implementation used CoreMotion device motion to shift album art and Ken Burns slow zoom/pan — all removed.

**How to apply:** If considering any motion-based visual effect on album art or backgrounds, skip it. Static or simple fade transitions are preferred.
